# Bypass Paywalls for Scientific Documents

## Description
This user script allows you to bypass paywalls for scientific documents by downloading them from Sci-Hub. It adds download buttons on Google Scholar, Scopus, and Web of Science, providing free access to scientific papers.

## Table of Contents
- [Installation](#installation)
- [Usage](#usage)

## Installation
To install the user script, follow these steps:
1. Install a user script manager like Tampermonkey or Greasemonkey.
2. Add the script to your user script manager.

## Usage
1. Navigate to a supported platform (Google Scholar, Scopus, or Web of Science).
2. The script will automatically add download buttons for accessing documents via Sci-Hub.

## Features
- **Automatic Domain Update**: Checks and updates the Sci-Hub domain if it changes.
- **Download Buttons**: Adds buttons on supported platforms to download documents directly from Sci-Hub.
- **Multi-Platform Support**: Works on Google Scholar, Scopus, and Web of Science.

## Supported Platforms
- **Google Scholar**: Adds download buttons for scientific papers.
- **Scopus**: Integrates download buttons for full-text access.
- **Web of Science**: Provides download buttons for accessing documents.
